# 🧠 Cancer Care Detection System (ML Web App)

An AI-powered web application that predicts the likelihood of breast cancer using machine learning techniques. The system takes medical input features and provides real-time predictions through an interactive web interface.

---

## 🔍 Overview
This project focuses on building an end-to-end machine learning pipeline, from data preprocessing and model training to deployment using a web framework.

---

## ⚙️ Features
- Predicts breast cancer based on input medical parameters  
- User-friendly web interface built with Flask  
- Real-time prediction using trained ML model  
- Efficient model integration using serialized (pickle) files  

---

## 🧪 Technologies Used
- Python  
- Flask  
- Scikit-learn  
- Pandas, NumPy  
- HTML, CSS  

---

## 🤖 Machine Learning Model
- Algorithm: Random Forest Classifier  
- Data preprocessing: Feature scaling and cleaning  
- Model training and evaluation using Scikit-learn  

---

## 📊 Workflow
1. Data collection and preprocessing  
2. Feature selection and scaling  
3. Model training using Random Forest  
4. Model evaluation  
5. Deployment using Flask web application
